## About

The Margin is a personal website covering:
- LLM research and AI
- Mathematics and computer science
- Personal reflections and ideas
- Theology

## Quick Start

```bash
# Clone the repository
git clone https://github.com/jonxpr10/the-margin.git
cd the-margin

# Install dependencies
pnpm install

# Start development server
pnpm dev
```

## Development

- `pnpm dev` - Start development server with hot reload
- `pnpm build` - Build for production
- `pnpm preview` - Preview production build locally
- `pnpm test` - Run tests
- `pnpm format` - Format code

## Tech Stack

- **Framework:** [Quartz](https://quartz.jzhao.xyz/) - A fast, batteries-included static site generator
- **Content:** Markdown with YAML frontmatter
- **Hosting:** GitHub Pages
